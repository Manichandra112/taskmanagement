# TaskFlow - Modern Task & Subtask Management

TaskFlow is a premium, high-fidelity web application built for seamless hierarchical task tracking, subtask scheduling, and team workload management.

## 🚀 Key Features

* **Hierarchical Tree View**: Create multi-level subtasks nested under parent containers, with collapsible/expandable nodes.
* **Smart Ancestor Recalculation**: Top-level parent statuses are computed dynamically based on the state of their active subtasks.
* **Custom Dropdowns**: Seamless, beautiful, animated custom dropdown selectors replacing native browser select inputs.
* **Assignee Management**: Sleek inline assignee renaming and list table management.
* **Quick Creator**: Add assignees on-the-fly directly inside the task creation modal.
* **Polished Micro-animations**: Sleek CSS visual transitions, animated hover cards, and custom vector icons.
* **Dashboard Snapshots**: Fast workload health summaries and delivery stats.
* **Flicker-free silent refreshes**: Bypasses page reloads for action states to keep UI transitions fast.

## 🛠️ Project Setup

### Backend (Express & SQLite)
1. Navigate to the `backend` folder:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev API server:
   ```bash
   npm run dev
   ```

### Frontend (React & Vite)
1. Navigate to the `frontend` folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Launch the hot-reloading dev server:
   ```bash
   npm run dev
   ```
